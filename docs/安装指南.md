本文档面向国内开发者，涵盖 Windows 和 macOS 环境下的 Git、Node.js、Python3 纯命令行安装与配置。所有工具均可通过脚本一键安装，无需手动下载。

## 前置要求

### Windows：启用 winget

Windows 10 1803+ / Windows 11 自带 winget。若未安装，先更新系统或从 [Microsoft Store](https://apps.microsoft.com/detail winget) 安装「应用安装程序」。

### macOS：安装 Homebrew

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

> 若 Git 访问慢，先配置代理或镜像（见下方国内镜像配置章节）。

## Git 安装

### Windows

```powershell
winget install Git.Git
```

### macOS

```bash
brew install git
```

### 验证

```bash
git --version
# git version 2.45.0
```

## 国内镜像配置

安装完成后，运行以下脚本一键配置国内镜像：

### Windows PowerShell

```powershell
# GitHub 加速
git config --global url."https://ghproxy.com/".insteadOf https://github.com

# npm 淘宝镜像
npm config set registry https://registry.npmmirror.com

# pip 清华镜像
pip3 config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple
```

### macOS / Linux Bash

```bash
# GitHub 加速
git config --global url."https://ghproxy.com/".insteadOf https://github.com

# npm 淘宝镜像
npm config set registry https://registry.npmmirror.com

# pip 清华镜像
pip3 config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple

# conda 清华镜像（仅 Anaconda 用户）
conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/pkgs/main
conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/pkgs/r
conda config --set show_channel_urls yes
```

## VPN 代理配置

若有 VPN，需配置终端代理：

### Windows PowerShell

```powershell
# 假设 VPN HTTP 代理端口为 7890，SOCKS5 为 7891
$env:http_proxy="http://127.0.0.1:7890"
$env:https_proxy="http://127.0.0.1:7890"
$env:all_proxy="socks5://127.0.0.1:7891"

# 永久生效（需管理员权限）
[System.Environment]::SetEnvironmentVariable("http_proxy","http://127.0.0.1:7890","User")
[System.Environment]::SetEnvironmentVariable("https_proxy","http://127.0.0.1:7890","User")
```

### macOS / Linux Bash

```bash
# 假设 VPN HTTP 代理端口为 7890，SOCKS5 为 7891
export http_proxy="http://127.0.0.1:7890"
export https_proxy="http://127.0.0.1:7890"
export all_proxy="socks5://127.0.0.1:7891"

# 永久生效
echo 'export http_proxy="http://127.0.0.1:7890"' >> ~/.zshrc
echo 'export https_proxy="http://127.0.0.1:7890"' >> ~/.zshrc
echo 'export all_proxy="socks5://127.0.0.1:7891"' >> ~/.zshrc
source ~/.zshrc
```

### Git SSH 代理

若使用 SSH 方式克隆仓库（`git@github.com:...`），需配置 SSH 代理：

```bash
# 编辑 ~/.ssh/config
Host github.com
    HostName github.com
    User git
    ProxyCommand connect -S 127.0.0.1:7891 %h %p
```

> Windows 用户编辑 `C:\Users\你的用户名\.ssh\config`，若文件不存在则新建。


```bash
brew install git node python@3.11
git config --global url."https://ghproxy.com/".insteadOf https://github.com
npm config set registry https://registry.npmmirror.com
```
